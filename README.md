# myBlog

## System:

* Ruby version: ruby-2.5.1
* Rails version: Rails 5.1.6
* Database: PostgreSQL Client version: 9.5.15

### How to run project

* Use rvm
```
rvm use
```
* Setup bundler
```
gem install bundler
```
* Run:
```
bundle install
```
```
rake db:create
```
```
rake db:migrate
```
```
rails s
```
